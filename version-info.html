OOPS! Evaluator

A plug-in for the Protégé Desktop ontology editor software that evaluates ontologies to detect the use of bad practices The plugin is made available as a workspace tab, which can be activated via the "Windows > Tabs > OOPS! Evaluator" menu item.

Features:

  • Evaluates ontologies to detect the use of bad practices(pitfalls).
  • Highlights items with pitfalls in the class, object properties and data properties hierarchies.
  • Shows details of the pitfalls of the selected item in the mentioned hierarchies.
  • Lets the user filter the pitfalls by an explicit list or by selecting a predefined category.
  • Lets the user fix the detected problems directly in the plugin's tab.
WARNING: This plugin requires internet connectivity, as it makes requests to a web service for the ontology evaluation task.

Acknowledgement

This plugin uses the OOPS! OntOlogy Pitfall Scanner! online tool for the ontology evaluation task. Specifically, this plugin uses its web service.
